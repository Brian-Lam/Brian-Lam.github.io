Projects

Here you will find a list of projects that I've worked on. Some of these were projects that I've made at work that I was given permission to list publicly, but most are projects that I've done in my free time.

College Personality Profiler

This is data aggregator that creates a personality profile out of a college based off of publicly available data. This was written primarily in PHP during the WUHack 2014 Hackathon. It pulls data from multiple APIs such as Facebook Graph API, Yik Yak posts, and Instagram, and to build data about the "average" person in the area, and presents it in a user-friendly format.

Collaborators: Ben Stolovitz, Morgan Redding, Hunter Wills, Audrey Cole

Technologies Used: PHP, Apache Web Server, Python, HTML, CSS

Asset Crawler

A python script that crawls a specific domain and keeps track of which CSS and Javascript assets are used the most. This script was created to determine which commonly used files should be concatenated and minified to speed up page load time.

Technologies Used: Python

CSS Conflict Parser

CSS Conflict Parser parses CSS files and keeps track of which styles are applied to selectors, then generates a report of conflicting styles applied to selectors, as well as non-conflicting files. Currently In Beta.

Technologies Used: Python
